
# 💍 Engagement Invitation Website

## 🚀 Overview
A beautifully designed digital engagement invitation website with elegant UI and smooth animations.  
Crafted to create a memorable and interactive experience for sharing special moments online.

---

## 📖 Description
This website showcases an engagement invitation with a modern and aesthetic design.  
It includes smooth scrolling, animations, and interactive elements for a premium feel.  
Built with clean UI/UX principles ensuring responsiveness, elegance, and performance.

---

## ✨ Features
- Responsive design (mobile + desktop)
- Smooth animations (Framer Motion)
- Navigation to section (scroll effect)
- Hovering effects (buttons, cards)
- Countdown timer support
- Elegant typography & UI consistency
- Interactive CTA buttons

---

## 🧩 Sections

- **Hero:** Displays couple names with elegant intro and engagement announcement  
- **Blessings:** Shows spiritual message and blessings for the couple  
- **Couple:** Introduces bride and groom with family details  
- **Event Details:** Shows date, time, and location of the event  
- **CTA Section:** Invites guests to join and celebrate  
- **Footer:** Contains credits and closing message  

---

## 🛠 Tech Stack
- Next.js  
- Tailwind CSS  
- Framer Motion  
- shadcn/ui
